# 测试问题集合

用于提示词工程训练和智能体效果测试。

## 测试问题 1：软件项目需求分析

```text
我想做一个校园二手交易平台，应该怎么做？
```

预期智能体应输出：

- 原始问题诊断；
- 缺失信息分析；
- 优化后的高质量问题；
- 目标用户；
- 功能模块；
- 非功能需求；
- 技术实现建议；
- PDF 报告。

---

## 测试问题 2：在线学习 App

```text
我想做一个在线学习 App，帮我设计一下。
```

预期智能体应输出：

- 学习场景分析；
- 用户角色；
- 课程、练习、测评、AI 辅助功能；
- 页面模块；
- 数据模块；
- 风险和改进方向。

---

## 测试问题 3：学生成绩分析系统

```text
我想做一个学生成绩分析系统，帮我规划一下。
```

预期智能体应输出：

- 成绩数据来源；
- 分析指标；
- 可视化模块；
- 教师端和学生端功能；
- 隐私和安全风险；
- 验收指标。

## 测试问题 4：科研选题优化

我想研究大模型和教育，帮我想一个选题。

预期智能体应输出：

- 研究问题优化；

## 测试问题 5：考试复习规划

```text
我快考试了，帮我复习高级算法。
```

预期智能体应输出：

- 复习目标；
- 知识点优先级；
- 高频题型；
